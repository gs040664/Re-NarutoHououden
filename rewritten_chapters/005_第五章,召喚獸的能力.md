# 第五章：召喚獸的能力

時間如流沙般從指尖悄然滑過。轉眼間，鳳凰寺悠羽來到這個世界已經一年有餘。

六歲的她，如今的身型雖依舊纖細，但已經退去了最初那種營養不良的乾癟。黑色長髮被打理得柔順光亮，黑紫色的大眼睛透著一股遠超同齡人的成熟與睿智。在「木葉湯」溫泉旅館，她已經從一名瑣碎的小雜工，被破格提拔成了負責調配人手的「小領班」。

老闆娘對這個撿來的「寶貝」讚不絕口，逢人便誇自家出了個天才管家。這導致悠羽走在街上買菜時，總能迎來鄰里大嬸們熱切的目光與善意的打趣。

然而，沒人能想到，這個笑容甜美、做事麻利的小女孩，此刻纖細的四肢上正負載著總計兩百斤的特製鉛塊。

「基礎……還不夠。必須把身體錘鍊到能承載更強大查克拉的地步。」

除了繁重的工作與基礎體能體修外，悠羽將所有的精力都投向了那一張沉潛著無數祕密的通靈契約捲軸。通過名為「靈識」的意識投影，她的意識頻繁穿梭於現實與鳳凰聖地「仙雲山」之間，在那片雲霧繚繞、靈氣逼人的神境中，認識了數位性格迥異的通靈夥伴。

首先是仙靈鳥「水樹」。那是一隻通體潔白、羽毛如瓷般細膩的奇特鳥類，雖然性格極其囉唆且喜歡反覆囉嗦，但卻擁有一項名為「仙靈羽衣」的神奇天賦。當牠憑附在悠羽身上時，悠羽可以完美地幻化成任何普通鳥類，甚至能掩蓋住身為人類的氣息。

其次是老仙鶴「神樂」。牠的氣息古樸得近乎陳腐，行事一板一眼，卻能引動仙雲山的靈性雲氣。那種雲氣對生者而言極其危險，能瞬間讓人迷失自我的心智，甚至引導靈魂直接脫殼而出。

當然，還有那位最為熟悉的武士影疾，牠依舊忠誠且凌厲，擅長大範圍的風遁切割與高速載運。

而在偵察領域，雨燕「五月」則是悠羽最得力的助手。那是一隻性格活潑、聲音甜美的小燕子，論飛行的靈活度與瞬時速度，牠在整個鳳凰一族中也位居前列。

至於幻術支援，則由「時音」與「阿吉」這對百靈鳥夫妻檔負責。妻子時音是個熱衷於家常理短的主婦，而丈夫阿吉則遺傳了如某些天才忍者（鹿丸）般「怕麻煩」的頹廢性格，但在編織複雜幻境時，牠們的默契幾乎無人能破。

「呼……終於成功了。」

在修行的第十五個月，悠羽在一次深夜的冥想中，清晰地聽到了腦海深處傳來的一聲輕微脆響。

那是八門遁甲的第一門——「開門」。

隨著這層枷鎖被徹底粉碎，一股前所未有的蓬勃查克拉如決堤的洪水般在她的經絡中奔騰不息。那種身體機能被瞬間推向極限的充盈感，幾乎讓她沉醉。

「體力與能量儲備翻了將近三倍……」悠羽握緊拳頭，感受著那股在指縫間跳躍的力量，眼中閃過一抹精光，「雖然總量上依舊遠比不上那些『怪物』，但在微觀控制上，我絕不會輸給任何人。」

憑藉著這種精細的操控力，悠羽在隨後的半年中開發出了屬於她的第一個原創戰術——「櫻幻瞬步」。

這是一項結合了瞬身術、替身術與初級幻術的複合技。在發動瞬間，她會通過微量的查克拉擬造出漫天飛舞、閃爍著淡淡螢光的櫻花瓣。這些絢爛而虛幻的景觀能在瞬息之間癱瘓敵人的視覺重心與注意力，而就在那一毫秒的空檔內，悠羽早已憑藉著爆發性的速度，轉移到了敵人的視野盲點，或早已遁入黑暗之中。

「雖然只是基礎術式的疊加，但在實戰中，往往越簡單的誘惑越致命。」

她對著庭院中那棵在月色下搖曳的古老櫻花樹，再次進入了修行的循環。

雖然深知自己現在與卡卡西、宇智波鼬那種天之驕子尚有雲泥之別，也沒有豪門大族的血繼限界作後盾，但那又如何？

既然沒有神格的眷顧，那便將凡人的基礎推向神跡的境界。

在這種近乎偏執的勤學苦練中，一名意外的「導師」候選人，正悄然向火之國邊境的這家溫泉旅館靠近。
