# 第八章：告別與出行

清晨，當火之國邊境的城市還在沉睡時，悠羽已經如往常般完成了五公里的晨跑。

即便昨晚因為興奮與不安交織而導致失眠，這具日漸強韌、被查克拉不斷滋養的嬌小身軀，卻依然保持著驚人的活力。她抹去額尖的汗水，心跳微微加快——一年的打工修行即將劃下句點，她正盤算著該如何正式向老闆娘請辭。

畢竟這一年多來，老闆娘對她的照顧，早已超出了單純的雇傭關係。

當她回到旅館大門時，卻撞見了令她猝不及防的一幕。

旅館全體員工、雜習、廚師，甚至連經常在門口掃地的老僕人，都整齊地列隊站在大門兩側。老闆娘站在最前方，目光慈和；而自來也撇著嘴，披著那一身豔紅色的短褂，正背著手倚在旁邊的柱子上，似乎早已等候多時。

「大家……都在這兒做什麼？」悠羽愣住了，原本準備好的請辭辭令全卡在了嗓子裡。

「要走了……對吧。」老闆娘輕聲開口，語氣中帶著一抹早已洞察一切的瞭然。

「嗯……」悠羽感覺喉嚨裡像是塞了一團沾水的棉花，鼻尖一陣酸澀。

就在一分鐘前，她還在心裡雀躍地高喊「偉大的忍者冒險終於要開始了」。可直到這一刻，看著那扇寫著大大的「油」字的燈籠，看著那些平日裡和她打鬧、教她偷懶、塞給她熱包子的同伴，她才猛然發覺——

她本以為這裡只是個暫時的避風港、一個攢錢和自習的驛站。

但事實上，她早已在這裡找到了在這個世界的「家」。這種久違的安心感與歸屬感，是任何高級忍術與神話召喚獸都無法取代的。

她被自己的「孤獨」蒙蔽了太久，直到要離去時，才驚覺這份羈絆早已深深刻進了靈魂。

「早就知道妳這小丫頭留不住，『小忍者』……」

千夜子眼眶微紅，快步走上前，將一個沉甸甸的旅行背包和一個精緻的絲質包裹塞進悠羽懷裡。

「這是行李，我昨晚都幫妳收拾好了。還有……」千夜子深吸一口氣，將那個包裹壓在悠羽手上，強顏歡笑道，「這是我們大家湊錢給妳準備的離別禮物，不許拒絕！」

隨後，這位豪爽的前輩轉過頭，對著自來也兇巴巴地揮舞著小拳頭：「喂！色鬼大叔！要是敢讓我們家悠羽跟著妳受苦，或者是讓她少了一根寒毛，我們全旅館的人可是會聯手發布賞金任務追緝你的！」

四周的員工們也紛紛憤慨地附和，自來也只能雙手作投降狀，苦笑著連連保證：「絕對不敢，絕對不敢……」

老闆娘走近，像一年前初次錄用悠羽時那樣，溫柔地順了順她那頭烏黑的長髮。

「去闖蕩吧。妳是有大志向的孩子。但也要記住，累了的時候，隨時都能回來。這裡的大門永遠會為妳敞開。」

悠羽的眼淚終於潰堤。

儘管體內住著一個二十七歲、歷經社會現實的成熟靈魂，但在這份不求回報的淳樸善意面前，所有的「理智」與「演技」都顯得如此蒼白。

她不再壓抑那股湧上心頭的酸楚，使勁地、深深地朝著大家鞠了一個躬，聲音哽咽卻大聲地喊了出來：

「這一年來……謝謝大家的照顧！悠羽一生都不會忘記老闆娘您的恩情……也不會忘記大家的！」

她哭得淚流滿面，甚至連鼻涕也毫無形象地流了出來。千夜子在一旁也忍不住轉過頭去抹眼淚：「這孩子……第一次看妳哭成這樣，害我也鼻酸了……」

「那麼……我出門了！」

悠羽再也不敢多待一秒。她用袖子狠狠胡亂一抹臉上的淚水與鼻涕，抱著行李，在眾人的告別聲中，頭也不回地朝著城門奔去。

她怕自己只要慢下一步，那份好不容易建立起來的「決意」就會被這份溫暖擊碎。

「走好啊——！」
「要記得回來看看——！」

身側的自來也步履穩健，悠閒地跟在一路狂奔的小女孩身旁。

直到跑出了城郊，悠羽的心情才漸漸平穩下來。她擦乾了最後一絲淚痕，眼圈雖然還有些發紅，但神態已恢復了初見時的那份清冷與傲氣。

感覺到自來也那充滿戲謔與探究的目光一直停留在自己臉上，悠羽老臉一紅，有些不自然地撇過頭，沒好氣地道：「大叔，你看夠了沒？」

「沒什麼。只是覺得……妳哭格的樣子比平時那副臭屁樣要『可愛』那麼一點點。」自來也嘿嘿一笑，眼底卻滿是讚賞。

悠羽為了化解這份尷尬，立刻切換到「修行模式」。

「師父，什麼時候開始教我忍術？」

「喔？妳想學什麼？」

「全部！」悠羽眼神熾熱，語氣認真得讓自來也有些頭疼，「不管是那個頭髮變刺蝟的《針地藏》，還是那個能把地變成稀泥的《黃泉泥沼》，只要是你的本事，我都要學！」

自來也不禁流下一滴大冷汗。這小鬼，倒是識貨得很。

他看著眼前這個背著比自己還大的背包、眼中閃爍著野望的小女孩，再次仰天大笑。

「既然心這麼大……那就從最嚴苛的『邊境修行』開始吧！在那之前，先把妳那雙沉得要命的負重鉛塊加重一倍再說！」

夕陽（朝陽）將一高一矮的影子拉得很長。火之國的國道上，這場即將改變歷史的「師徒之旅」，正式邁出了第一步。
